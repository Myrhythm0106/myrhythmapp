## Test Scenarios

### Scenario 1: New User - Guided Path
**Goal:** Test the complete onboarding flow for first-time users

**Steps:**
1. Open app in incognito/private browser
2. Click "Get Started"
3. Select user type (e.g., "Memory Support")
4. Complete the welcome flow
5. Select a plan and start trial
6. Complete the LEAP assessment
7. Arrive at dashboard

**What to check:**
- [ ] Is the flow intuitive?
- [ ] Are instructions clear?
- [ ] Does progress save if you leave and return?
- [ ] Any confusing steps?
- [ ] Time to complete (target: < 5 minutes)

---

### Scenario 2: New User - Explorer Path
**Goal:** Test users who skip guided onboarding

**Steps:**
1. From landing page, click "Explore" or navigate directly to /dashboard
2. Dismiss any prompts
3. Try to use features without completing onboarding

**What to check:**
- [ ] Are features accessible?
- [ ] Are there helpful prompts to complete setup?
- [ ] Does the app gracefully handle incomplete profiles?

---

### Scenario 3: Memory Bridge Core Flow
**Goal:** Test the primary value feature

**Steps:**
1. Go to Memory Bridge (/memory-bridge)
2. Click "Start Recording"
3. Speak for 30-60 seconds, including:
   - "I need to call Dr. Smith tomorrow at 2pm"
   - "Don't forget to pick up milk"
   - "I promised Sarah I'd send the photos"
4. Click "Stop Recording"
5. Wait for processing
6. Review extracted actions
7. Confirm actions and add to calendar

**What to check:**
- [ ] Does recording start/stop clearly?
- [ ] Are action items correctly extracted?
- [ ] Is the confidence score reasonable?
- [ ] Do actions appear in calendar?
- [ ] Is audio quality feedback provided?

---

### Scenario 4: Brain Health Reminders (NEW)
**Goal:** Test the new reminder escalation system

**Steps:**
1. Create an action item with a due time in 5 minutes
2. Wait for the first reminder (Level 1 - Gentle)
3. Don't respond - wait for escalation
4. Note the reminder at Level 2, then Level 3
5. Test snooze functionality
6. Test the "Done" button
7. Check if Support Circle is notified (if configured)

**What to check:**
- [ ] Do reminders appear at correct times?
- [ ] Does escalation work (gentle ‚Üí moderate ‚Üí urgent)?
- [ ] Are contextual memory anchors helpful?
- [ ] Does snooze work correctly?
- [ ] Is Support Circle notified at Level 3?

---

### Scenario 5: Daily Brain Boost (NEW)
**Goal:** Test the daily challenge feature

**Steps:**
1. Navigate to dashboard
2. Find the Daily Brain Boost widget
3. View today's challenge
4. Click "Why this helps?" button
5. Mark challenge as complete
6. Check if streak updated
7. Return tomorrow and check for new challenge

**What to check:**
- [ ] Is the challenge clearly explained?
- [ ] Does "Why this helps?" provide useful info?
- [ ] Does completion update the streak?
- [ ] Is there variety in challenge types?
- [ ] Does the widget look good on mobile?

---

### Scenario 6: Support Circle Setup
**Goal:** Test inviting support members

**Steps:**
1. Go to Support Circle settings
2. Click "Add Member"
3. Enter a test email and name
4. Set relationship and permissions
5. Send invitation
6. Check for email delivery
7. Accept invitation (use test email)
8. Verify member appears in circle

**What to check:**
- [ ] Is the invitation flow clear?
- [ ] Does email arrive promptly?
- [ ] Are permissions properly explained?
- [ ] Can member see appropriate information?
- [ ] Is the dashboard useful for support members?

---

### Scenario 7: Calendar Integration
**Goal:** Test calendar functionality

**Steps:**
1. Create manual calendar event
2. Add reminders to event
3. Create event from Memory Bridge action
4. View day, week, and month views
5. Edit an existing event
6. Delete an event
7. Check reminder notifications

**What to check:**
- [ ] Is event creation intuitive?
- [ ] Do different views work correctly?
- [ ] Are reminders triggered at right times?
- [ ] Can events be edited easily?
- [ ] Does mobile view work well?

---

### Scenario 8: Promise Score Tracking (NEW)
**Goal:** Test the commitment tracking feature

**Steps:**
1. Complete several Memory Bridge recordings
2. Extract and confirm actions
3. Mark some actions as complete
4. Check Promise Score in dashboard
5. View Promise Score history
6. Mark an action as "Not Doing"

**What to check:**
- [ ] Does Promise Score update correctly?
- [ ] Is the score calculation clear?
- [ ] Does completing actions increase score?
- [ ] Is feedback encouraging?

---

### Scenario 9: Returning User Experience
**Goal:** Test the experience for existing users

**Steps:**
1. Log out and log back in
2. Check if data persists
3. Verify streak status
4. Check for "What's New" notification
5. View the What's New page
6. Verify all settings saved

**What to check:**

- [ ] Is all data preserved?
- [ ] Are streaks calculated correctly?
- [ ] Is What's New notification visible for new features?

## Feedback Survey Questions

After testing, please answer these questions:

1. **Ease of Use (1-5):** How easy was the app to navigate?

2. **Value (1-5):** Would this app be useful for someone with memory challenges?

3. **Design (1-5):** How would you rate the visual design?

4. **Speed (1-5):** Did the app feel fast and responsive?

5. **Memory Bridge:** Was action extraction accurate?

6. **Reminders:** Were the escalating reminders helpful or annoying?

7. **Daily Brain Boost:** Would you complete these challenges daily?

8. **What's missing:** What feature would you add?

9. **What's confusing:** What was hardest to understand?

10. **Would you pay ¬£10/month?:** Yes/No/Maybe - why?
