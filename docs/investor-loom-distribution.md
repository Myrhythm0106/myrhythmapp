# Investor Loom — Distribution Plan

**Companion to:** [`investor-loom-script.md`](./investor-loom-script.md)
**Owner:** Founder · **Metric:** Loom's built-in per-viewer analytics

---

## 1 · Where the Loom lives

- **Primary:** Loom, unlisted link, comments off, download off. Loom is the primary because per-viewer view analytics beat YouTube's aggregate counts for warm outreach.
- **Mirror:** Unlisted YouTube upload of the same MP4 for anyone who blocks Loom at work. Description contains only the Loom link.
- **Never:** posted on LinkedIn, Twitter/X, or embedded on the marketing site during the Founding phase. This is a hand-shared artefact, not a broadcast.

Store the Loom URL, YouTube URL, and MP4 backup in the founder's password manager under `MyRhythm — Investor Loom v1`.

## 3 · Tracking

Loom's per-viewer view analytics **are** the metric. No extra tooling.

Weekly (Monday review):
- Count of new viewers, split by audience (cold / warm / clinician). Loom tags each viewer by email of the recipient.
- Number of viewers who watched past 3:30 (traction segment). This is the qualified-lead signal.
- Reply rate by audience. Target: warm ≥ 40%, cold ≥ 10%, clinician ≥ 60%.

Add a single line to the weekly log in `docs/90-day-sprint.md` §8: `Loom sends: N · past-3:30: N · replies: N`.

---

## 4 · Re-record triggers

Do not fiddle. Re-record only when one of these lands:

- LOI 1 signed → update the traction segment (3:30–4:15).
- Founding Member count crosses a round number (50, 75, 100) → update the traction segment.
- The Ask number changes → update the closing segment.
- Discharge Bridge Kit copy changes materially → update the demo segment.

Keep the last two versions in cloud storage. Never delete.

---

## 5 · What we are not doing

- Not paying for Loom Pro custom-branded video hosting until we have 3 LOIs.
- Not making a "public" version of this video for the marketing site during v0.1.
- Not A/B testing thumbnails. One founder, one face, one thumbnail.
- Not writing a longer 10-minute version. If a viewer wants more, that is the 25-minute live call.
